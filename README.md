# Python Generators in C

## Description
A header-only library of generators in pure C99, modified from https://github.com/tsoding/coroutines.

Currently only works for x86_64.

## Quick Start

```console
$ make -j8 all
$ ./build/coprimes
$ ./build/fib
$ ./build/lexer
$ ./build/primes
$ ./build/square
```

See [./examples/](./examples/) folder.
